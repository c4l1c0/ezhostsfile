# ezHostsFile
This program is used to unblock sites by adding their ip and url to /etc/hosts file.

**Currently only works on Linux OSes**

## Dependencies
* make
* gcc (>=4.5)
* libcurl

## Compilation
```
make
```

## Usage
```
ezhostsfile [options] <url>
```

for more info, run:

```
ezhostsfile -h
```
